# 0003 — 寻龙诀规则契约与冲突裁决

* **日期**:2026-06-27
* **状态**:已采纳
* **背景**:`docs/superpowers/plans` 与知识库 `30_Knowledge/Dragon-Seeking/03-规则体系` 存在多处口头化、边界歧义的规则表述(早盘板 `<10:00` vs `<=10:00`、最高连板 0/1/6+ 无状态、F18 缺 20%-30% 区间等)。本仓库此前没有版本化的规则契约,导致盘后与盘中逻辑各自复制一份。本 ADR 把 `.omx/plans/dragon-seeking-rule-gap-transformation.md` 第 3 节的冲突裁决固化为不可静默修改的契约。

## 结论

### 规则版本
- 所有确定性规则以 `rule_version = "dragon-formula/1.0.0-draft"` 标识,定义于 `config/rules/dragon_formula_v1.toml`。
- 任何规则语义变更必须升级 `rule_version`,禁止用同一版本号改变历史含义。
- 原因码只能追加或版本化,不能改同一代码的语义。

### 冲突裁决(写入契约,后续调整需升版本)

| 冲突 | 裁决 | 理由 |
|---|---|---|
| C 级"禁买"与模板"由用户拍板" | C、D、B− 不进入推荐候选,可进入观察/解释记录 | F17 与风控体系更新、更明确 |
| F19"3 天 2 板"与 F14"4 天 2 板" | 两信号并存:F19 用近 3 个交易日 >=2 板作准入共振;F14 用近 4 个交易日 >=2 板作评分 1.10 倍加成 | 两条规则目的不同,不互相覆盖 |
| 早盘封板 `<10:00` 与脚本 `<=10:00` | F19 使用严格 `first_seal_time < 10:00:00`;恰好 10:00:00 不算早盘共振 | 按 F19 原文执行并消除边界歧义 |
| F18 缺 20%-30% 区间 | 新增 `DEFENSIVE`:允许观察,风险预算系数 0.5,不输出积极买入措辞 | 保守填补空档,参数可配置 |
| 最高连板只定义 2-5 板 | `<=2` 归 FROZEN,3 压制,4 活跃,`>=5` 主升 | 单调外推,避免 0/1/6+ 无状态 |
| 10/20/30cm 未覆盖 ST 5cm | 加入 ST 5cm,但 ST 先被 F19 排除;5cm 仅用于历史样本归一化 | 保证历史计算正确 |
| "候选最多 5 只"与 ML 全量样本需求 | 全量写 `candidate_observations`,仅通过硬门槛且排序后的前 5 写兼容表 `candidates` | 同时满足训练完整性与推荐约束 |

### 不可破坏约束(与 AGENTS.md 一致)
1. `candidates.score` 字段名与 0-150 语义不变。
2. 全量首板观察样本与最终发布 Top 5 分表保存。
3. 数据源失败不得伪装成真实 0 值或空市场。
4. 非交易日、关键数据无效时必须 fail closed。
5. 所有门槛返回机器可读原因码。
6. 盘后与盘中不得各自复制 F17/F18/F19 逻辑。
7. migration 兼容已有 `recap.db`,可在副本演练与回滚。
8. AI/UZI 只解释,不得覆盖硬过滤、排名、交易日、数据质量与风控结论。

### 未最终确认参数(保守默认,保持配置化与灰度)
见方案第 25 节:F18 20%-30% 风险预算 0.5、`>50%` aggressive 1.25 仅作标签、股性 UNKNOWN 覆盖率达标后降级观察、50m-100m 封单默认不扣分(由 `weak_seal_penalty_enabled` 控制)、adjusted score 回测达标前不替换排序主键、持仓数据由外部传入。

## 影响
- 新增规则文件 `config/rules/dragon_formula_v1.toml` 与 `config/lhb_seats_v1.toml` 为唯一规则来源。
- 盘后 `src/recap_engine.py` 与盘中 `src/data_pipeline/` 必须消费同一规则契约与同一套纯函数。
- 知识库 `history_resonance.py`、`lhb_quality.py` 中的硬编码截止日期与绝对路径仅作算法思想参考,不得直接搬入生产。

## 验证
- `tests/test_rule_contract.py` 覆盖配置加载、阈值单调性与原因码稳定性。
- `tests/test_golden_candidates.py` 覆盖 2026-06-19/24/25/26 金样本。
